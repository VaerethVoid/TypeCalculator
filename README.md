# Pokémon Void Type Calculator

An interactive type weakness calculator for the fan-made Pokémon fangame **Pokémon Void**.

## Features
- All 20 types including custom types **Cosmic** and **Light**
- Select up to 2 types for dual-type calculations
- Instant results showing 4×, 2×, 1×, ½×, ¼× and immune (0×) matchups
- Embedded on the [Pokémon Void Wiki](https://pokevoid.miraheze.org)

## Usage
Visit the live calculator at:
`https://vaerethvoid.github.io/TypeCalculator/type_calculator_interactive.html` when it is done.
